# Project 

Doge Fitness

## Description

Fitness application which lets the user make a weekly workout plan.

### Dependencies

* .NET 5

* .NET 5 Runtime

* Node.JS

* SQL Server

* Visual Studio Code

### Executing program

Rodrigo.Tech.Fitness.Web.API 

```
dotnet run
```

rodrig-tech-fitness-ui 

```
npm start
```
